# 修路理论与人情维护

> 「大多数人只会在掉坑里时才哭爹喊娘，但聪明人知道在晴天时把伞送出去。」

## 什么是修路理论

人脉关系不是等你需要时才去挖的井，而是平时就要织的网、修的路。四年不联系的人，关键时刻想开口都腿软——你只在需要时才想到对方，对方也能感觉到。

**类比**：农民种地，播种浇水除草占 99% 的功夫，丰收挥镰反而是最轻的一下。人情世故也一样，功夫全在平时。

## 人情破冰六步

当一个很久没联系的人你需要重新接上时：

1. **按死求人的念头**——先别想自己要什么，先想欠了对方什么
2. **发感恩短信**——重提旧事 + 功劳归他 + 表达感恩（不是乞求，是成就对方）
3. **长期维护**——特产拜访（坐10分钟就走）、请教反馈、日常关心——绝口不提帮忙
4. **通过介绍人**——越过中间人直接联系是给自己埋雷，先跟介绍人知会一声
5. **让对方主动开口**——等他问「最近有什么难处」
6. **水到渠成**——对方轻轻一抬手，事就成了

## 核心原则

- **把人脉从一次性买卖拆成过日子**：让他习惯生活里有你，不觉得你是资源，你俩是有交情的
- **感恩信的底层逻辑**：不是讨好，是给对方「当伯乐的顶级情绪价值」
- **双节点维护**：帮你的 + 介绍人，两头都要维系
- **船得提前10年造**：永远别等要过河了才找船
