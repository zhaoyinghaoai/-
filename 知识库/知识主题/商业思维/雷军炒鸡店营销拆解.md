# 雷军炒鸡店营销打法拆解

> 用雷军的产品思维做中小餐饮——7步从0到周边三公里第一，不花一分钱广告费。

## 核心观点

**产品是及格线，让别人知道你产品好才是核心竞争力。**

很多人觉得雷军靠营销，其实他的逻辑是：先把产品做到及格线以上，再用最笨的方法把口碑做起来。你缺的不是好味道，是让别人知道你味道好的方法。

## 7步打法

| 步骤 | 做法 | 关键点 |
|------|------|--------|
| 1. 爆款先行 | 只留一款招牌炒鸡+3个凉菜，死磕细节 | 鸡精确到3斤2两，爆炒12分钟不多不少 |
| 2. 先圈顾客再开业 | 开业前建群，每天发选材/配方/门头投票 | 开业前已有几百个精准粉丝 |
| 3. 开业直播 | 站在厨房门口直播，公开成本结构 | 一只鸡成本48，卖68，利润20 |
| 4. 实在变参数 | 把厨房里的标准全公开，做成海报和短视频 | "藏在厨房里的实在一文不值" |
| 5. 全员拍抖音 | 每个员工都有自己的抖音号 | 1000播放奖10元，1万播放奖100元 |
| 6. 限量造稀缺 | 前20份8折，最后5份送菜 | 越有时间限制，人越愿意早来 |
| 7. 异业合作 | 周边超市/理发店/水果店互换代金券 | 零成本互相交换客户 |

## 提炼的商业原则

1. **少即是多**——砍掉多余产品，集中精力做好一款
2. **先有用户再有店**——开业前就圈粉丝，不是等开业再找顾客
3. **透明即信任**——公开成本、公开标准，顾客觉得放心就会来
4. **让实在被看见**——不是你实在就够了，要让客户看得见
5. **借力打力**——不花钱请网红，让员工和周边商家帮你传播

## 金句

> "藏在厨房里的实在一文不值，只有让客户看见的实在才值钱。"

> "做中小餐饮不是比谁能吃苦，是比谁更会把苦变成钱。"
