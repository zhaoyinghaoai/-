# 参与感营销六步法

> 雷军用性价比赢信任，马云用参与感赢认同。把决策权分给顾客，让顾客从「来吃的」变成「来选的」——最终变成「不舍得让它倒闭的人」。

## 六步框架

| 步骤 | 核心动作 | 底层逻辑 |
|------|---------|---------|
| 1. 选品权给顾客 | 开业前投票定菜单，票高免费吃 | 从「我卖什么」变「你想吃什么」 |
| 2. 帮邻居赚钱 | 让隔壁面馆来店里做招牌面不收租 | 利他=利己，邻居老客变你的新客 |
| 3. 预定代替备货 | 每天投票明天菜，多做票高的 | 零库存风险，需求驱动生产 |
| 4. 差评变口碑 | 公开整改对比图+每月最佳差评请吃饭 | 差评是免费改进信号不是敌人 |
| 5. 顾客替你说话 | 外卖卡片拉人、好评上电视、周排行榜 | 把顾客变销售员 |
| 6. 店变大家的店 | 顾客故事墙+每月见面会+活页菜谱 | 情感绑定→顾客不舍得你倒闭 |

## 与传统营销的对比

| 维度 | 传统营销 | 参与感营销 |
|------|---------|-----------|
| 关系 | 你卖我买 | 我们一起做 |
| 决策权 | 老板说了算 | 顾客投票定 |
| 传播 | 花钱打广告 | 顾客主动传播 |
| 护城河 | 产品好 | 关系深+情感绑定 |

## 浩总启发

参与感的核心不是「让顾客参与」这句话本身，而是**真的把决策权分出去**。对于浩总的场景：
- 军事文章：让粉丝投票下一期分析哪个热点
- PPT副业：让客户参与大纲设计
- 快递站：让顾客投票取件时间/增值服务
