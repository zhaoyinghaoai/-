# 闲鱼上的闷声发财小生意

> 闲鱼不只是二手平台，更是普通人靠解决"小麻烦"赚钱的地方。

## 两个案例

| | 改PDF | 整理聊天记录 |
|------|------|------|
| 需求 | 字体对不上/版面乱/水印 | 合同/业务记录提交公司或律师 |
| 定价 | 20-50元/单 | 30-100元/单 |
| 效率 | 几分钟一单 | 十几分钟一单 |
| 收入 | 一天几十单，月入过万 | 一天好几单 |

## 共性

- 客单价低、需求琐碎、频次高
- 用户自己搞特别费劲，会做的人几分钟搞定
- **用户买的不是技术，是省下的时间和精力**

## 核心方法论

1. 不需要高大上的技术壁垒，比用户多懂一点点就够了
2. 把流程跑通，做成标准化交付
3. AI工具降低交付成本，但用户需求还在，愿为结果付费
4. 一个人一台电脑就能开始

## 行动路径

1. 逛闲鱼/淘宝，看销量高但不起眼的服务
2. 问身边人最近有什么琐碎又不得不做的事
3. 学会用AI工具组合，形成标准化交付流程
4. 先从一个具体小需求开始，积累几十个客户再放大

## 金句

个体创业的起点，往往就是从帮别人解决一个10块钱的具体小麻烦开始的。
