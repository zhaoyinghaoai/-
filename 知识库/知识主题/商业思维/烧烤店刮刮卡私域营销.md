# 烧烤店刮刮卡私域营销

> 「现在做餐饮靠平台引流，就是给平台打工。把客人攥在自己手里，生意才能越做越稳。」

## 来源

## 核心观点

1. **刮刮卡零门槛获客**：进店加微信就送刮刮卡（奖品全是菜品/啤酒，非现金），不强制消费，客人不抵触，加微信比例高达90%。3天上千人进店。

2. **三层锁客设计，层层递进**：
   - 第一层：刮刮卡零门槛加微信（获客）
   - 第二层：每桌每次限兑1张 + 有效期1个月（锁回头）
   - 第三层：99元小额储值送68元硬菜（锁长期，散客变会员）

3. **四种群运营盘活私域**：
   - 每晚8点红包手气王 → 送10串/半打啤酒（激活群+拉到店）
   - 晒单返10元券 → 老客宣传+锁下次到店
   - 周三会员专属日 → 专属套餐比到店便宜10-20%
   - 老带新裂变 → 拉3个同城朋友进群送菜

4. **免费裂变传播**：中大奖拍抖音/朋友圈带地址 → 再送5瓶啤酒。零广告费，顾客主动帮你打本地广告，一传十十传百。

5. **成本控制和合规底线**：七成小奖 + 两成中奖 + 一成大奖，总成本控制在营业额5%以内；别搞现金大奖、别暗箱操作、别大额储值（99元小额安全）；规则不能破——每桌每次限兑一张是锁复购的核心，搞套路客人信任直接崩。

## 适用范围

不仅限于烧烤店，火锅、家常菜、面馆都能用。核心是可复用的获客-锁客-裂变三段论，不依赖平台流量。

## 对浩总的启发

这套玩法的精髓是**不花广告费把客人圈到自己手里**。浩总做军事文章也面临类似问题——平台给流量就有，不给就没了。如果能把读者导入私域（微信/微信群），用类似逻辑激活和维护，或许能突破82粉的瓶颈。
