# 赚钱四维框架：刚需 × 信息差 × 流量 × 情绪

> 所有赚钱的生意都可以拆成这四个维度。它们不是四个赛道，是可以叠加的维度。维度越多，赚钱越稳。

## 四维详解

### 1. 刚需——不买不行
- **定义**：用户必须消费的领域（吃住行医教），需求永不消失
- **核心能力**：稳定供给，不是创新
- **风险**：竞争激烈，利润薄
- **例子**：早餐摊、快递代收、药店

### 2. 信息差——你知道别人不知道
- **定义**：认知不对称带来的变现空间
- **核心能力**：学习能力 + 传播能力
- **误区**：信息差≠骗人，是把真实有价值的认知卖给需要的人
- **例子**：代运营、技能培训、渠道代购

### 3. 流量——注意力就是钱
- **定义**：聚集一批人的注意力，然后变现
- **核心能力**：内容输出 + 精准定位
- **关键**：1000个精准用户 > 100万泛粉
- **例子**：短视频、公众号、小红书博主

### 4. 情绪——让人买到想要的感觉
- **定义**：卖的不是功能，是感觉（身份感、安全感、希望感）
- **核心能力**：洞察人性需求
- **特点**：最暴利，溢价空间最大
- **例子**：奢侈品、知识付费、高端咖啡

## 叠加公式

| 维度组合 | 实例 | 威力 |
|---------|------|------|
| 刚需 + 信息差 | 职业培训（教人考证） | ⭐⭐⭐ |
| 刚需 + 情绪 | 月子中心（刚需+安全感） | ⭐⭐⭐ |
| 流量 + 情绪 | 网红带货（注意力+冲动） | ⭐⭐⭐⭐ |
| 信息差 + 流量 | 知识博主（认知+传播） | ⭐⭐⭐⭐ |
| 刚需 + 信息差 + 情绪 | 教育/知识付费 | ⭐⭐⭐⭐⭐ |
