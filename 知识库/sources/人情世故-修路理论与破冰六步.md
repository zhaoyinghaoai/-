# 人情世故：修路理论与破冰六步

> 人脉不是等你渴了才挖的井，而是你几年前就必须开始织的网。平时「修路」，关键时刻才有路可走。

## 核心观点

1. **修路理论**：大多数人只会在掉坑里时才哭爹喊娘，但聪明人知道在晴天时就把伞送出去。4年不联系的人，关键时刻想开口都腿软——因为你只会走路，从不修路。

2. **破冰第一步：按死求人的念头，先还情**。你现在开口叫讨债，得先把四年欠的情包装成惊喜还回去。发一条感恩短信：重提旧事 + 功劳归他 + 表达感恩——这不是送礼，是提供「当伯乐的顶级情绪价值」。

3. **把人脉从一次性买卖拆成过日子**：逢年过节带特产坐10分钟就走；隔一两月请教专业问题并反馈思考；关注对方头疼脑热孩子升学——但绝口不提「帮忙」二字。让他习惯生活里有你这个懂事的晚辈。

4. **双节点维护**：想进庙，先跟守门的知会一声。越过介绍人直接联系是给自己埋雷。先看望当年的长辈，说「多亏您和那位领导，我想去看看人家又怕太唐突」——他不但不拦，还会手把手教你怎么敲门。

5. **农民种地的类比**：播种浇水除草占99%，丰收挥镰反而是最轻的。人情世故也一样，功夫全在平时。个人奋斗是地基，人情世故是减少阻力、跨越阶层的捷径。

- 「让生米煮成熟饭——让他不觉得你是他的资源，你俩是有交情的」
- 「船得提前10年造」
- 「在晴天的时候把伞送出去」
