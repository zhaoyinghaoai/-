# 高考满分作文：《东风的一生》

> 以「东风」第一人称视角，写尽风的一生——从无名之风到被人赋予意义，再到人间长出自己东风。核心洞察：风从没变过，变的是人。

## 核心观点

1. **人赋予意义，风只是经过**：同样一阵风，李商隐看是「东风无力百花残」（衰败），辛弃疾看是「东风夜放花千树」（繁华）。人不看风，人看的是自己。

2. **东风=人的愿望**：离别时叫离别，欢喜时叫欢喜，失败时叫时运，成功时叫天意。人把自己无法解释的事交给风。东风其实是一种人的愿望。

3. **从等待到创造**：当人们无力时等待东风，当人们勇敢时创造东风。结尾点出：年轻人不再等待东风，他们奔跑、学习、创造——人间长出了自己的东风。

4. **万物只是流转**：群山没有你我，羊群继续低头吃草。也许只有人类才执着于意义，万物只是流转，风只是经过。

5. **写作技法亮点**：第一人称拟人叙事 + 古诗典故密集引用 + 时空跨度大（从古代到现代火箭「东风」） + 结尾强力升华。

## 关键概念/金句

- 「人并不看风，他们看的是自己」
- 「东风其实是一种人的愿望」
- 「当人们无力时，他们等待东风；当人们勇敢时，他们创造东风」
- 「风从来没有改变过，改变的是人」
- 「后来人间长出了自己的东风」
