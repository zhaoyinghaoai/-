# 马云开快餐店：参与感打法的六步

> 雷军用性价比赢信任，马云用参与感赢认同。把决策权分给顾客，让顾客觉得「这家店是我看着做起来的」。

## 核心观点

1. **把选品权交给顾客**：开业前投票定菜单，被选最多的免费试吃——顾客从「来吃的」变成「来选的」
2. **帮隔壁老王也赚钱**：让邻居面馆来店里做招牌面不收租金，帮邻居赚钱→他的老客来你家→他推荐你家——用利他换复利
3. **预定代替备货**：每天群里投票明天菜→票数高的做，低的不备→零库存风险
4. **差评变口碑**：每条差评亲自回复+门口贴整改对比图（左边「肉太少」→右边「加量30%」）——差评是免费的改进信号
5. **让顾客替你说话**：外卖放卡片拉人进群，好评上电视屏——把顾客变成销售员
6. **把店变成大家的店**：顾客故事墙+每月见面会+活页菜谱留改进建议——当顾客觉得「这家店是我看着长大的」，他就不舍得让它倒闭

## 关键概念/金句

- 「当一个人觉得'这家店是我看着长大的'，他就不舍得看着它倒闭」
- 参与感=分权给顾客，从「你卖我买」变成「我们一起做」
- 帮邻居赚钱的底层逻辑：利他即利己
