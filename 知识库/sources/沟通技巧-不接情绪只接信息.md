# 沟通技巧：不接情绪，只接信息

> 对方说了一句让你不舒服的话，拆开看只有两个东西——情绪和信息。你只需要做一件事：不接情绪，只接信息。一接情绪就吵，一接信息就聊事情。

## 核心观点

1. **核心方法**：当对方的话让你不舒服，先别回，想一下这句话是「情绪」还是「信息」。比如「你怎么这么慢」——情绪是「我很急」，信息是「我想知道还要多久」。不回情绪，只回信息。

2. **四个实战案例**：
   - 领导质疑产出 → 「您具体指哪个部分？我好调整」——把审判变成请教
   - 合作方说方案不一样 → 「哪部分跟之前不一样了？」——不解释中间需求变了很多次
   - 朋友抱怨你不理他 → 「你是不是想我了？」——把抱怨翻译成想念
   - 饭局有人阴阳怪气 → 「你觉得最考验什么能力？」——把阴阳怪气变成提醒

3. **心理学原理：防御性归因**。被质疑时人的第一反应是自我保护和解释反击，但一旦防御，主动权就交出去了。不接情绪只接信息，本质是把对方的攻击性语言重新定义为需求表达。

4. **效果链条**：不防御 → 不内耗 → 从容 → 主动权在自己手里。以前别人一句质疑立刻解释越抹越黑，现在听到的是需求不是情绪，不需要防御。

5. **通用规律**：大部分让你不舒服的话，拆开看只有情绪 + 信息。你帮对方把情绪翻译成需求，同时把对话节奏抢回自己手里。

## 关键概念/金句

- 「不接情绪只接信息」
- 「把审判变成请教」
- 「把抱怨翻译成想念」
- 「不防御就不内耗，不内耗就从容，从容的人才有主动权」
