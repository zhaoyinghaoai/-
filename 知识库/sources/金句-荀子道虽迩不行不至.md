# 道虽迩，不行不至；事虽小，不为不成

> 荀子两千年前就把「执行力」说透了——再近的路不走也到不了，再小的事不做也完不成。

## 核心观点

1. **行动是最大的变量**：距离和难度不重要，「是否迈步」才是唯一决定结果的因素
2. **小事的复利**：不为不成——大事由小事积累，不做的代价不是零，而是错过
3. **金句的百搭性**：个人成长类、创业类文章结尾均可使用，用古人智慧收束现代故事

## 关键概念/金句

- 道虽迩，不行不至；事虽小，不为不成
- 千里之行，始于足下（《道德经》）
- 不积跬步，无以至千里（《荀子·劝学》）
