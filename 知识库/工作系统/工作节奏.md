# 工作节奏

> 浩总的时间分配和黄金工作时间说明。

## 一天的时间结构

| 时间段 | 内容 | 备注 |
|--------|------|------|
| 07:00 | 快递点开门 | 开始营业 |
| 07:00 - 12:00 | **黄金工作时间** | 上午相对空闲，有1小时以上整块时间 |
| 12:00 - 19:30 | 快递点营业 | 有客人来取件时处理，间隙可碎片化记录 |
| 19:30 | 快递点关门 | 一天结束 |
| 晚上 | 家庭时间 | 两个女儿由妈妈和媳妇带，浩总可参与陪伴 |

## 关键特征

- **黄金时间在上午**：这是浩总一天中唯一能专注做自己的事的时段
- **手机为主**：碎片化记录、灵感捕捉主要在手机上完成
- **电脑为辅**：深度整理和结构化在电脑上做
- **每天1小时+**：有整块时间，可以做有深度的整理和复盘

## 建议的工作节奏

| 做什么 | 什么时候做 | 用什么设备 |
|--------|-----------|-----------|
| 灵感捕捉 | 随时 | 手机 → Raw/Inbox/ |
| 深度学习/整理 | 上午黄金时间 | 电脑 |
| 文章写作 | 上午黄金时间 | 电脑 |
| 复盘 | 每天结束前 / 每周固定时间 | 电脑 |
| 碎片阅读 | 快递点间隙 | 手机 |
