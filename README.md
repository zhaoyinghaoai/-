# 浩总的个人知识库

> 一个能持续生长的本地知识库，基于 [Karpathy LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) 三层架构。

## 快速理解

这个 Vault 分三层：

```
原始材料/  → 你丢进来的原始材料（文章、截图、笔记、语音转文字）
知识库/    → AI 帮你整理后的结构化知识（你阅读，AI 维护）
产出/      → 你拿出去用的东西（文章成品、PPT、计划书）
```

**核心理念**：你负责输入和提问，AI 负责整理和维护。知识会随着使用越来越丰富。

## 怎么用

### 日常使用

1. **随手记** → 任何东西先丢进 `原始材料/收件箱/`，不要求分类
2. **让AI消化** → 告诉AI"处理一下收件箱"，AI 会读取、提炼、更新到知识库
3. **读知识库** → 打开 `知识库/Index.md`，从这里开始浏览你的知识库
4. **做产出** → 需要写文章、做PPT时，从知识库里取素材，成品放 `产出/`

### 手机端

- 主要入口：`原始材料/收件箱/`
- 随手截图、语音转文字、一段话，直接丢进来
- 不需要当场分类，AI 会帮你处理

### 电脑端

- 深度整理、写文章、做PPT
- 浏览和编辑知识库页面
- 定期复盘（`产出/复盘/`）

## 关键文件

| 文件 | 用途 |
|------|------|
| `AGENTS.md` | AI 操作手册——告诉AI怎么维护这个 Vault |
| `知识库/Index.md` | 知识目录——从这里导航整个知识库 |
| `知识库/Log.md` | 操作日志——追溯所有变更 |

## 知识主题

`知识库/知识主题/` 下有6个主题区：

- 军事 — 军事热点、武器装备、地缘政治
- AI — AI 工具、AI 创业、自动化
- PPT — 设计技巧、排版、配色
- 育儿 — 育儿知识、女儿成长
- 个人成长 — 自我管理、习惯养成、心态建设
- 商业思维 — 商业认知、创业方法论、变现策略
